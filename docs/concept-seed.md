# 世界观种子 ·《碑外问道》（暂名）

> 作者向的概念输入源（喂给 `playbook/1-world-create.md`）。参考《剑来》之神、原创其形。
> 架构师据此展开为 `world/` 六份设定；本文件是源头，改概念先改这里。

## 一句话立意
一个被判了「至贱命格、注定一事无成」的拓碑孤儿，偏要在一个「认出身、拼气运、讲规矩」的天下，
靠「守本心、占个理」，一步一步问出自己的道、挣出活人的尊严。

## 三旋钮（本次定稿默认值）
- **节奏**：紧凑——保留剑来味（人情/哲思/慢热底色），但爽点更密、推进更快。
- **格局**：单主角主线为主；中后期向「剑垣群像」延展（预留接口，不强上）。
- **基调**：热血励志为主，冷峻为底——强者有立场、规矩有重量，但不致绝望。

## 五要素
### 类型
东方玄幻 · 修行文明流 · 多天下格局。市井人情 + 修行升级 + 庙堂权谋 + 剑修热血，
底色是「规矩与道理之争」的东方哲思。单主角贴身视角、群像为辅。

### 时代
上古一统之世「圆同天」毁于**断鳌之劫**，万载后碎成数座彼此隔绝又暗通的「天下」。
当下正值衍洲天下「**三千载封山和议**」末期：旧序礼崩、气运将洗牌、荒外妖族叩关日紧；
而养了千年气运的 **沉鳞洞天 · 碑亭镇** 封印将开。故事始于开封前夜的碑亭镇。

### 核心冲突（四层）
1. 人 vs 命（情感内核）：至贱命格的少年逆天改命，靠本心而非天赋。
2. 规矩 vs 道理（主题引擎）：规矩由强者书写、未必公道；主角在「按规矩」与「讨个理」间立心。
3. 气运争夺（外部驱动）：洞天开封，各方为气运而来，小镇成棋盘，少年是最关键的棋子。
4. 文明 vs 荒外（宏大背景）：剑垣以剑修血肉挡妖族，正统文明存续之战，终与少年命运交汇。

### 力量体系方向
主路双修（练气/术士 · 武夫/体修）+ 三支线（剑修 · 读书人文运 · 山水神祇）
+ 元规则层（**气运·命格** 硬通货、**规矩·道则** 由神祇执行、**本命物** 命门）。
哲学引擎：规矩未必是道理，但修行先修心。详见 `world/power-system.md`。

### 主角初设
**沈砚**，碑亭镇拓碑孤儿，明面命格判词「五行驳杂、命格至贱、背字之命」。
守拙、认死理、对公道执拗，信条「凡事得占个理」「不急，一步一步来」。
起手：一双草鞋、半方磕角旧砚、能吃苦的好筋骨；镇口瞎眼守碑老人半个照看。
**🔒 隐藏钩子（作者知、主角不知）**：至贱命格系被人故意改判镇压，真相关乎沉鳞洞天千年气运
与一缕上古「问道碑」碑魂，开封之夜引爆。建角色时入 `knowledge_boundary.must_not_know`。

## 与《剑来》异同
借神：讲规矩/问道理内核、气运命格硬通货、山水神祇幽冥秩序、剑修守城悲壮、蝼蚁靠本心逆天、慢热人情。

剑垣≠剑气长城、沉鳞洞天≠骊珠洞天、沈砚≠陈平安。

## 五问定稿（OPEN_QUESTIONS 默认值，可改）
1. **本命物**：沈砚日后本命物 = 祖传**磕角旧砚**为媒介，内蕴一缕**碑魂 / 本命字「拙」**（与命格钩子绑定）。路数：凡俗起步、筋骨宜武夫，剑修/文运皆留口，不写死。
2. **改判者阵营**：把他命格压成「至贱」的，是**封局的三教大佬之一**，借**司命体系**之手改判镇压（具体是谁=后期谜题）。
3. **封局真意**：沉鳞洞天表里两层——表「养气运、备战荒外」，里「守碑魂、养一个活体碑灵之锚」，而沈砚正是被选中（或意外成为）的那个锚。
4. **释教（寂照禅宗）**：暂作远方背景，不入场（保聚焦衍洲/剑垣、单主角紧凑）。
5. **女主/反派**：女主方向＝**剑垣的剑修少女**（剑来宁姚味，原创）；近线反派＝**为开封气运而来、牵涉改判内幕的某方爪牙**。二者各自待 `/character-create` 建档。
