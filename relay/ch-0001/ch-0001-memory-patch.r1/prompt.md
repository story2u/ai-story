# 第一章 至贱命格

午后的日头毒辣，照在无字碑林的青石板上，白花花一片。

沈砚蹲在一块半人高的残碑前，左手压纸，右手拈着沾墨的毡刷，一上一下，匀匀地拓。碑面没有一个字——无字碑林全是这样的石头，大大小小几百块，有的比人高，有的只到膝头。过路的修士偶尔出几个钱，让镇里人替他们把碑面拓下来带走，说是“留个碑形印证”。

至于是印证什么，没人问，也没人说。

沈砚拓到第三张时，余光里扫见几个人影从碑林入口拐进来。他手上没停，只把毡刷落得慢了些。等那几个人走近了，他才认出当头的是在镇衙当差的持册镇差，身后还跟着两个帮手，一个捧着簿册，一个腰里别着铁尺。

那镇差在碑林边站定，摊开簿册，扯着嗓子念了个名字。

“沈砚——”

毡刷停了。

沈砚先把未干的拓纸小心压平，才直起身。他没有立刻应声，目光从那镇差脸上移到那本簿册上——那是清册，开封前例行的复核。每年都要来这么一回，但往年没点过他的名。

“至贱命格，五行驳杂，”那镇差念完判词，把簿册一合，声音比刚才还大了几分，“清册复核期间，至贱命格不得滞留碑林。沈砚，把随身旧砚交了入册查验，即刻暂离碑林。这是上头的吩咐。”

围观的人渐渐聚拢。碑林边本来人就不少，有路过的、有在阴凉处歇脚的，听见“至贱命格”四个字，目光都朝沈砚这边扎过来。

沈砚没有退。

他把毡刷搁在墨碟边沿，直起身，向那镇差行了一礼，声音不高，字字压稳：“差爷要查册，我认。可旧砚是我吃饭的家什，不是碑林公物。要入册查验，请说清是哪一条旧规，谁押印，何时归还。凡事，得占个理。”

那镇差没想到他会回嘴，愣了一息，才皱眉道：“开封是大事，清册复核谁都得认——你一个至贱命格的，还敢跟镇里叫板？”

“我没叫板。”沈砚的声音仍不高，“我只是问一句章程。”

他说这话时，身子微微挡在桌前——那半方磕了角的旧砚就搁在纸墨旁边，灰扑扑的，不起眼，却是他从头到尾没让人碰过的东西。

人群边上，一个游方修士打扮的中年人抱臂看了一阵，这时悠悠往前挪了两步。

“哟，这位小哥好硬气——‘凭哪条旧规、谁押印’，说得好，说得好哇。”

那人穿着一件洗得发白的旧道袍，腰间挂了个葫芦，脸上挂着和气的笑。他走到僵局的斜角处站定，既不靠镇差，也不靠沈砚，一副“嘴痒多一句”的散漫样子。

“不过嘛——”他拖长尾音，侧身向那镇差假笑，“差爷，这碑林的清册是有年头的规矩了，这小哥问得也不算无礼——索性，您把是哪位大人经手押印说一说，大伙儿心里也踏实，您说是不是？”

那镇差被将了一军，脸色有些挂不住，只反复说“上头吩咐的”“清册复核谁都得认”。两名帮手往前半步，像是要给沈砚施压。

那外乡修士又转向沈砚，眼角笑着：“当然了，旧砚嘛，我瞅着也是件用惯了的老物件儿，差爷看验一眼也就完了——小哥，您这砚，拓了几年了？”

沈砚心里一紧。

这人看似在帮腔，问的句句却在往旧砚上引。他没有立刻作答，先朝那修士略一颔首算是致意，礼数到了便收回目光。

“这砚随我拓碑，已有七年。”他说，“风霜旧物，不值几个钱，却是我吃饭的家什。差爷要看验，我仍是那句话：请先说清押印与归还。若只看一眼，便请当众看，当众记，当众还。”

他的声音仍低，却比方才更清楚，让在场的人都听得见。

围观人群里响起零碎的议论。有人在说“这少年倒是不怯”，也有人嘀咕“至贱命格还敢这么硬，不怕吃官司”。沈砚把那些话都听在耳里，面上一丝没露。

那镇差被逼到墙角，帮手们面面相觑。拿不出具体条文，又当着这么多人下不来台。

那外乡修士又开口了：“差爷，我多句嘴：这碑林清册旧契，历来是哪位镇老经管押印？您手下若一时寻不着，让小哥把砚台搁碑座上，您二位当众记个形制、拓个印文，回头查明了契文再还——东西又没长腿跑，大伙儿都瞧着，您面子上也过得去，您说呢？”

那镇差脸色变了又变，最后哼了一声：“行，要查章程是吧？那就去镇衙旧契房，当面验明清册旧规。你——带着旧砚，跟我走。”

他让帮手先拦住沈砚离开碑林的路，改口要沈砚一起去旧契房。围观的人见热闹还没完，三三两两跟在后头。

沈砚弯腰收拾拓具。他先把未干的拓纸小心卷好，再把墨碟、毡刷一一收进布袋，最后才拿起那方旧砚——指尖碰到磕角时顿了一息，随即用旧布裹紧，贴身收好。

那外乡修士仍站在碑林边，目送沈砚跟着那镇差走远。他脸上还挂着笑，眼底的光却已经收了。

“沈砚、旧砚、七年拓碑——”他低声念了一遍，背着手，慢悠悠地也往镇衙方向去了。


镇衙旧契房在镇子东头，一间灰扑扑的老屋，满架积灰的卷宗，窗纸黄得透不进光，只靠门口一豆油灯照明。

沈砚在门槛内两步停住，目光扫过那一排排柜架。尘味扑面，混着陈年竹纸的涩气。

那镇差一进门就催：“要查什么赶紧，别磨蹭。”

沈砚没有急着翻柜。

“旧契分甲乙丙三库，”他说，语气平得像在说一件再寻常不过的事，“甲库是近三十年的镇务登记，乙库是碑林专档，丙库是更早的散卷。清册押的是碑林做活的章程——那该走乙库。烦请差爷开乙库柜，我指卷。”

那镇差愣了：“你怎么知道……”

“跑腿听熟的。”沈砚接得很快。

那外乡修士不知什么时候也跟到了门口，靠在门框上，一副顺路看热闹的样子。听见沈砚说出三库分法，他眉梢微微一动，随即又恢复了那副笑嘻嘻的表情。

“哎呀，真是详尽的记载！这三库的分法，倒是少见的细节。”他轻声道，“敢问，您之前是何时、何处熟悉这旧契房的？”

沈砚没有立刻回答。他先看了那修士一眼，又收回目光，对那镇差说：“七年里我替人拓碑，拓成的纸、领过的纸墨、补过的押记，都要经旧契房的手。柜名是跑腿时听熟的，不是偷看来的。”

那镇差哼了一声，让帮手去叫账房。

老账房被唤出来时还揉着眼睛，听那镇差说明了来意，捋着胡须看了沈砚一眼，没多说什么，只从腰间解下一串乌沉沉的钥匙。但他没有立刻开乙库，先摊出一册经手簿。

“要查卷，先写明清册号、经手人、查验缘由。”老账房声音沙哑，“小哥要指卷，也只能隔着柜门指，不能碰卷——且先说个大概年头或押记名目，我好翻。”

那镇差趁机把旧砚从桌边移到案心，由帮手盯住。他不再伸手夺，却把话头压成了另一层规矩：“若说不出可查的卷号与名目，旧砚便仍要按清册暂记待验。”

沈砚的指腹压在旧砚磕角上，没有立刻开口。屋里静了一息，只听得见灯芯偶尔毕剥一响。

他深吸一口气。

“差爷要名目，先查乙库‘碑林做活登记’、‘旧具随工留用’一类。”他开口了，声音不高，一字一字说得很稳，“年头约在七年前，我初入碑林扫洒之后。押记未必是我名下，或在收留孤子的那一笔里。”

他顿了顿。

“不急，一步一步来。烦请账房先生先按经手簿开目录，若无这项，我再答清册。但旧砚既已说当众待验，便请仍记在案——莫离桌、莫离眼。”

老账房看着他，浑浊的眼睛里闪过一丝什么。他没接话，只转身开了乙库的门。

乙库积灰更重。老账房屈身钻进去，灯影在架间晃动，尘絮浮沉。逐卷翻检的声音断断续续传出来——每出一卷，先抹灰，再蘸唇翻页，竹纸簌簌作响。

那镇差扶刀守在桌旁，目光不时扫过旧砚，指尖在刀柄上轻叩。

沈砚站在原处没动。他眼睛追着账房的手，耳朵听着翻页的节奏，呼吸压得很稳。那外乡修士靠在门框边，也没有再多话，只是目光偶尔在沈砚和旧砚之间来回一落。

约莫过了两刻。

老账房从乙库探出半个身子，手里夹着两卷残破的卷宗。他把卷宗放在案上，拍拍灰，拣出几页与经手簿比对。

“七年前收留孤子、支领纸墨、补立押记，确实有登。”老账房声音沙哑，“做工时日也吻合。但——”

他翻到一页，指腹在纸面上停了停。

“旧具来历栏空白，只记‘随工交接’。”

那镇差立刻凑近细看，眉头越皱越紧。他翻出清册指给账房看，老账房捋须摇头。两人低声说了几句什么，那镇差的脸色不太好看。

那外乡修士这时又开口了，声音和气：“镇差公，旧具一栏空白，却只记随工交接，这其中若有交接者的名号，倒是别有洞天。若方便，恳请稍作指点，谁曾将旧具递于此？”

他问得随意，像是在聊家常。但沈砚听出这句话里的钩子——这人还是在追旧砚的来路。

他没接话，只把目光钉在老账房手上。

老账房沉吟片刻，又翻出一卷——碑林做活登记底册。两卷并置，经手簿记“随工交接”，底册记“七年工期内因工留用”，日期、押记、纸墨支领条数一一吻合。

老账房抬眼，合卷。

“旧砚来历确有凭据，非贼赃，非私匿。”

那镇差脸色更难看了，但仍咬着不放：“旧契房只记用物去向，不议物主归属。”

老账房没理他，取了一张黄纸，提笔蘸墨，缓缓写下一份验状。写完，搁笔，念给屋里的人听：

“旧砚暂归沈砚照管，三日内持旧契至镇衙复核。复核前，旧砚不得离身、不得外借。沈砚亦不得擅离碑亭镇。”

他把验状推到桌案边缘，然后将旧砚推出半尺——

没有说归属，但已许暂留。

沈砚伸手拿回旧砚时，指尖是稳的。他把它用旧布裹好，贴身收起，然后向老账房行了一礼。

“谢过账房先生。”

老账房没看他，只摆摆手，径自收拾卷宗去了。

那镇差站在一旁，脸色铁青，却说不出更多话来。他带来的帮手也松了劲儿，互相看了一眼，退后半步。

沈砚没有多看他们，转身朝门口走去。经过那外乡修士身边时，他脚步微顿，余光扫见对方靠在门框上，指尖虚叩着门框边缘，目光在砚与账房间逡巡了一回。

两人没有说话。

沈砚迈过门槛，走入午后的日光里。


入夜，沈砚回到碑林边那间破屋，反身落闩。

他在门后站了一阵，竖起耳朵听外面的动静——虫鸣、风声、远处偶尔一两声犬吠，都正常。没有人跟来。

他这才把裹着旧砚的布包解开。拓纸、旧契凭据逐张理平，按要紧处折齐，压在手边不离开视线。

屋里只有一盏油灯，光晕黄而淡。沈砚坐到墙根，把旧砚拿在手里，指腹在磕角上停了一息。旧砚触手冰凉，与往日无异。

没事。

他呼出一口气，正要把旧砚收起——手指停住了。

他把旧砚重新放在掌心，慢慢握紧。

是温的。

不是被日头晒过的温度，而是从里往外透的一种微温，不烫手，却绝不是石头该有的温度。沈砚盯着它看了几息，将它用旧布裹紧，仍贴身收好。

他的目光落回拓纸上。

纸的颜色比方才暗了一层。不是灯影变化——他把油灯挪近又移远，纸色没有随之变动。那暗沉像是从纸里面洇出来的，某种说不清的、正在干涸的东西。

他指尖在纸面上方悬了半寸，没有碰。

就在这时，纸角无风自卷——微微扬起一个边，像被什么看不见的气流带了一下，随即平复。

沈砚屏住呼吸。

隔着窗，碑林方向传来极轻的一声石响——不像碎石落地，倒像有人翻了一页巨石的书。沉闷，遥远，只一声，再无后续。

沈砚等了一盏茶的工夫，什么也没再发生。

他把油灯挪回原处，先摸了摸窗缝——干的。又摸了摸墙根——干的。指腹划过桌——干的。不是雨，不是漏水，也不像灯影。

他再次看向那张拓纸。

纸面已经恢复平整，在灯下看起来与普通旧纸无异。他正要收起，目光却扫到纸右下角有一道不寻常的痕迹——

不是墨拓。是纸面上自然形成的压痕，像是当年拓碑时纸下压过什么极薄的东西。

沈砚把纸侧对灯光，凑近看。

凹痕最深的一段约半指长，弯如残月，两端渐浅直到消失。中间断开三处，每段缺口都不足一粒米的宽。纸背面相应位置没有任何凸起。凑到灯下细看，凹痕底部泛着极淡的铅灰色，像是什么矿物粉末渗进了纸纹。

不是潮，不是霉，不是寻常拓坏。

沈砚捏着纸最外沿，把它同旧契凭据分开平放。旧砚仍裹在怀里，他把拓纸挪到桌子的另一头，不让它靠近怀里的旧砚。

他坐在墙根，看着这两样东西，安静了很久。

白日的情形在脑子里慢慢过了一遍——那镇差当众念出“至贱命格”时的眼神，那外乡修士一句接一句看似帮腔实则试探的问话，旧契房里老账房翻出残卷时那镇差的紧张，还有验状上那句“三日内持旧契复核”。

今天这事没有完。

沈砚把拓纸按原折痕收好，单独压进一叠最不起眼的旧纸中间，避开旧契凭据，也不让它靠近怀里的旧砚。做完这些，他又起身摸了摸门闩，确认闩紧了，才重新坐回墙根。

灯芯烧到底，跳了两跳，暗了下去。

他没有再点灯。

眼睛适应了一会儿黑暗，窗缝里透进来淡淡的月光。他看向窗外——碑林的方向，夜色浓重，什么也看不见。

旧砚在怀里，拓纸在旧纸堆里。两样东西都有说不上来的古怪，可他一样也弄不明白。

不对。是不该在现在弄明白。

沈砚把呼吸放慢，闭了一会儿眼，又睁开。他记住拓纸凹痕所在的位置和三处断口的形状，在心里默念了两遍，然后不再想它。

明日先从旧契房的旧例里找说法。若能遇见守碑老人，就只旁敲侧击，不把纸和砚一并露出来。不急，一步一步来。


同一夜，碑亭镇另一头的客栈里，柯九正坐在灯下，面前摊着一本薄薄的笔记。

他手里拈着笔，笔尖悬在纸面上方，好一会儿没有落下。屋外更夫敲过二更，梆子声在夜风里传得很远。

他写下一行字：

“沈砚，碑林拓碑七年，随身旧砚，熟旧契房规矩。至贱命格——但命格这事，上头说过，不能只看面上判的。”

又写一行：

“旧具随工交接，经手栏空白。三日复核窗口可用。”

搁下笔，他把笔记合上，收入袖中暗袋。

窗外的月光照进来，在他脸上落下一道不深不浅的影。柯九吹熄了灯，在黑暗里坐了一会儿，然后低声自语了一句，声音轻得只有自己能听见：

“这个少年，有意思了。”
