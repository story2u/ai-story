# 正文内时间轴（timeline）

> 仅记忆管理员可写。只记**开篇之后、正文中**发生的事（世界背景史在 `world/history.md`）。
> 连续性检查器据此校验事件先后、季节、年龄、时长。版本 1 ·《碑外问道》。

| id | t（可排序时间） | 章节 | 标签 |
|---|---|---|---|
| _（示例，开写后替换）_ evt-0001 | 封元 0 日 | ch-0001 | 开封前夜·碑亭镇 |

> t 用自定历法即可，例如以"开封之日"为第 0 日，记作「封元 N 日」。
