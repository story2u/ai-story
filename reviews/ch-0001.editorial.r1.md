## 4. 因果逻辑

**结论：pass。**

**主链（压迫→占理→悬疑）**

1. 开封前清册复核 → 当众点「至贱」+ 夺砚/驱离：stakes 明确，「往年没点过他」暗示背后有人借势（未说破，合理）。
2. 沈砚要章程 → 镇差答不出押印 → 柯九拱火 → 转旧契房：每一步有可见动机，非硬推。
3. 旧契房：沈砚引乙库 → 账房程序加码 → 残卷闭环「随工留用」→ 验状暂管 + 三日复核：胜负符合 brief「只退半步、危机未解」。
4. 夜间：白日旧砚被多方盯视 → 独处微温 + 拓纸异变 + 碑林石响：时间因果顺；沈砚先排查潮气/灯影再下结论，符合凡俗认知路径。

**无「为推进而推进」的断裂点**：镇差改口「去旧契房」、账房最终许暂留，均有场面压力与程序依据。
