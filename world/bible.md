# 世界观圣经（Bible）·《碑外问道》

> 给人读的世界总览 + 索引。结构化细节在同目录其它 markdown，本文件负责「定调」与「导航」。
> 仅世界观架构师可写。版本 1（据 `docs/concept-seed.md` 展开）。

## 一句话设定
> 上古一统之世「圆同天」毁于断鳌之劫，天地碎成数座天下。在讲规矩、拼气运、认命格的衍洲天下，
> 一个被判「至贱命格」的拓碑孤儿，偏要靠「守本心、占个理」，问出自己的道。

## 基调（本作定稿旋钮，可调）
- **节奏**：紧凑——保留剑来味（人情/哲思/慢热底色），但爽点更密、推进更快。
- **格局**：单主角主线为主；中后期向「剑垣群像」延展（预留，不强上）。
- **基调**：热血励志为主、冷峻为底——强者有立场、规矩有重量，但不致绝望。

## 世界三支柱（详见各文件）
- **铁则与禁忌** → `rules.md`：断鳌隔界、真死不复、山盟反噬、凡俗禁令、命格可改判（逆天大事）。
- **力量体系** → `power-system.md`：练气十境 / 武夫九境双主路 + 剑修·读书人·山水神祇三支线 +
  **气运·命格** 元规则。可判定：境界 order、压制关系、跨体系换算、越级上限俱全。
- **势力与天下** → `factions.md`、`geography.md`、`history.md`：三教（文庙/道门/禅宗）、大衍王朝、
  斩荒剑垣、山水衙门、荒外妖族；圆同天 → 断鳌 → 诸天下 → 三千载封山和议 → 当世开封前夜。

## 核心冲突（四层嵌套）
1. **人 vs 命**：至贱命格的少年逆天改命，靠本心而非天赋。
2. **规矩 vs 道理**：规矩由强者与神祇书写、未必公道；主角在「按规矩」与「讨个理」间立心。← 全书灵魂
3. **气运争夺**：沉鳞洞天开封，各方为气运而来，碑亭镇成棋盘，少年是最关键的棋子。
4. **文明 vs 荒外**：剑垣以剑修血肉挡妖族，正统存续之战，终与少年命运交汇。

## 开篇情境
故事始于衍洲东陲沧澜道、沉鳞洞天中的**碑亭镇**——一座被三教刻意封起来养了千年气运的小镇，
镇中有一片上古**无字碑林**。封印将到期开封，气运将外溢，群雄、宗门、神祇皆来「捞鱼」。

## 给各角色的提示
- **角色演员**：你只活在自己的认知里；这个世界的「规矩」处处都在，但「规矩」未必就是「道理」。
- **导演**：升级与揭秘节奏交替，别让主角太顺；让「占个理」既是主角的软肋，也是他的剑。
- **总编**：战力一律对照 `power-system.md` 的 order 与压制表机械判定；越级必须有登记依据
  （剑修越一境、武夫硬抗、山水神祇地利、气运护身——都要查得到出处）。
