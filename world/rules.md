# 世界铁则（rules）·《碑外问道》

> 不可违背的世界法则。仅世界观架构师可写。目的：让总编能**机械判定**情节是否违背设定。版本 1。

## 根本法则
| id | 法则 | 约束了什么 |
|---|---|---|
| rule-duanao | 断鳌之劫后，诸天下隔绝，「天弦虹桥」多已断裂 | 跨天下往来极难，需特定渡口/大代价；不得随意让角色在天下间来去 |
| rule-no-resurrection | 神魂俱灭、或本命物被毁者，真死不可复生 | 真死角色不得复活（夺舍/转世是另一码事，需登记） |
| rule-benming | 修士有与性命相连的**本命物/本命字**，是底牌也是命门 | 本命物碎 = 重伤乃至陨落；本命物须登记，不得临场新增 |
| rule-shanmeng | 对天地或山水神祇立下的**山盟誓约**，违则反噬（轻则废修、重则魂消） | 立过誓的角色违誓必须付代价 |
| rule-fanjin | **凡俗禁令**：修士不得在凡人聚居区肆意动用超过「凝罡/易筋」之力 | 违者折气运、遭山水衙门追责追剿——保护凡俗、约束强者 |
| rule-siming | 命格由幽冥**司命**体系记录；改判/镇压命格是逆天大事，有大代价 | 命格改判须有够格者出手 + 付代价，不可轻率（主角钩子之源） |
| rule-qiyun | 气运可积/夺/赌/借，但**夺人气运必招杀劫反噬** | 夺气运不是白拿，须有相应代价与因果 |

## 绝对禁忌（任何角色都不得让其发生，除非架构师显式修订本表）
- 无 `power-system.md`「合法出口」依据的**越 2 个 order 击杀**。
- 凭空出现未登记的功法、法宝、本命物、地名、天下。
- 角色在不可能的时间内跨越超出 `geography.md` 距离表的路程；或无渡口擅入别的天下。
- 已判真死者再次出场。
- 在凡人聚居区无视 `rule-fanjin` 大动干戈而无人追责。

## 设定变更流程（红线）
- **谁能改**：仅世界观架构师。
- **怎么改**：其他角色只能提交「变更申请」，由架构师 approve / reject / amend，并追加下方变更记录。
- 涉及命格改判、跨天下、复活类情节，必须显式走本流程。
