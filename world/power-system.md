# 力量体系（power-system）·《碑外问道》

> 仅世界观架构师可写。**可判定**是硬要求：每条路有 order、能力边界、突破条件、禁忌；
> 跨体系有换算、越级有上限——让总编能机械判断「战力崩坏 / 违规越级」。版本 1。

## 总览：两主路 + 三支线 + 一元规则
- 主路①**练气（术士）**：御灵气、术法神通，走十境天梯。
- 主路②**武夫（体修）**：纯粹炼体，「不讲道理只讲拳头」，走九境。
- 支线**剑修**：练气或武夫中专修剑道者，持本命飞剑，**战力可越一境**（招牌，有代价）。
- 支线**读书人（文运）**：三教之儒，养文胆，「以理服人」乃至「言出法随」，在讲规矩处可以小搏大。
- 支线**山水神祇**：土地→城隍→河伯→山神→岳渎，凭神位执掌**辖地内的规矩之力**。
- 元规则：**气运·命格**——定上限、可逆改、可夺赌借；关键时刻「该死不死」。

## 支线 · 三条

### 剑修
不是独立天梯，而是练气/武夫中专修剑道、凝**本命飞剑**者。招牌：**战力可越一境杀敌**
（如证道杀合真之下者不算稀奇），代价是耗本命、折气运、易入心魔。镇守斩荒剑垣的主力。

### 读书人（文运 · 文胆品阶）
不以杀伐论强弱，以文胆品阶论：**开窍 → 明理 → 养气 → 胸有惊雷 → 言出法随 → 立言（亚圣/圣）**。
养文气/正气，在文庙辖地、讲理论道、山水衙门断案等「规矩场合」可**以小搏大**，
高阶能「言出法随」短暂改写小规矩。出了讲理的场，战力大打折扣。

### 山水神祇（神位品秩）
**土地 → 城隍 → 河伯 → 山神 → 岳渎正神**。神位赋予**辖地内的规矩之力**：拘魂、立禁、断因果、
凭「规矩」短暂压制更高境的闯入者（地利）。一旦离开辖地，神力大减，近乎凡俗。受朝廷与文庙双重节制。

## 压制关系（总编战力判定核心）
**同体系**：
- 高 1 个 order = 显著压制（低者需机缘/拼命/算计方可一搏）。
- 高 2 个 order = 碾压（常规手段无解）。
- **越级上限**：常规不可越 **2 个 order** 杀敌。

**越级的合法出口（必须查得到出处）**：
| 出口 | 效果 | 前置 |
|---|---|---|
| 剑修本命 | 战力越 1 个 order | 角色为剑修、持本命飞剑；有耗损代价 |
| 武夫硬抗 | 同 order 近身压制练气士；可越境死战 | 角色为武夫 |
| 山水地利 | 神祇在辖地内短压更高境 | 在其辖地、动用规矩之力 |
| 文运借势 | 读书人在讲理场合以小搏大 | 文庙辖地/论道/断案等场合 |
| 气运护身 | 高命格者关键时刻「该死不死」 | 命格/气运登记在案（见下） |

**跨体系粗略换算**（混战时给总编一把尺，非精确）：
- 武夫 order N ≈ 练气士 order N 的战力，但**近身武夫占优、远程练气占优**。
- 读书人/神祇不入此尺，按「场合 + 辖地」判定其能否以弱胜强。

## 元规则 · 气运与命格
- **命格九品**（自下而上）：至贱 → 凡庸 → 显达 → 将相 → 国士 → 帝王 → 圣裔 → 真龙 → 天命。
  命格定**潜力上限与气运亲和**；高命格者修行顺、机缘多、关键时刻有气运护身。
- **命格可改判**：由幽冥「司命」体系记录；高人/神祇可暂改、镇压一个人的命格记录——**逆天大事，有大代价**。
  （主角「至贱」即被改判镇压，是全书核心钩子。）
- **气运三级**：个人气运 ⊂ 宗门/王朝气运 ⊂ 天下气运。气运**可积**（功德机缘）、**可夺**（杀劫夺舍）、
  **可赌**（问道赌斗）、**可借**（气运加身）。沉鳞洞天开封 = 一大笔沉淀千年的气运外溢，故各方来抢。

## 战斗禁忌（违 = 战力崩坏，总编拦截）
- 无上表「合法出口」依据的越 2 个 order 击杀。
- 未登记的本命物 / 功法 / 法宝临场出现。
- 凡人聚居区肆意动用超阶之力（违 `rules.md` 凡俗禁令，遭山水衙门追责、折气运）。
- 已判真死（神魂俱灭 / 本命物碎）者再出场。
